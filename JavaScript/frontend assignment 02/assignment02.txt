    HTML, CSS file and JS file must be separated into 3 files.
    The program should have at least 7 different elements e.g. buttons, text, list, div, submit
    All seven elements should 7 different events

    The 7 different events should manipulate 7 different divs or other html elements 
    Be creative, make something  you enjoy smiler
